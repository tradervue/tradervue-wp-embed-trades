Allows you to embed shared trades from Tradervue via shortcode.

== Description ==

This plugin will embed [shared trades](http://www.tradervue.com/shared) from Tradervue into Wordpress using a shortcode.

On Tradervue, shared trades have a URL like

http://www.tradervue.com/shared/trades/421249

To embed that trade into Wordpress, enter the following:

[trade id=421249]

== Installation ==

* Install using WordPress built-in Add New Plugin installer
* Activate the plugin.
